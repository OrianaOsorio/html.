HTML

Hyper Text Markup Language, más conocido como HTML es el lenguaje con el que se escriben las páginas web.
Las páginas web pueden ser vistas por el usuario mediante un tipo de aplicación llamada navegador, necesario para ver los resultados de nuestro trabajo. El navegador más utilizado en nuestro medio es google chrome.
HTML nos permite aglutinar textos, sonidos e imágenes y combinarlos a nuestro gusto.
